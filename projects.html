Remote Temperature monitoring and control

Overview

The project was aimed to monitor and control temperature and humidity remotely from anywhere which was accomplished by MQTT and MQTT over web sockets protocols to communicate with HiveMQ cloud.

Working

(1) The temperature and humidity readings read by ESP8266 were sent to the ThingSpeak cloud and also to the MQTT broker HiveMQ via publish method.

(2) The second ESP8266 which was the subscriber of the published topic recieves the message and displays it on 128 x 64 OLED module interfaced through I2C protocol.

(3) The responsive webpage hosted on GitHub servers made by front end technologies (HTML, CSS, BootStrap and javaScript DOM) was used to recieve the data from ThingSpeak and display the live data on widgets.

(4) Moreover the webpage was also connected to the HiveMQ to serve as MQTT javaScript web client via MQTT over websockets to turn the fan on or off.

IOT Based Irrigation System

Overview

The project was aimed to be implemented on large scale to automate the farming techniques and reduce the water loss and labour cost.

Working

(1) ESP8266 operating in access point mode was used as a main micro controller.

(2) Four D1 mini each connected to the soil moisture sensor was operated in station mode to wirelessly send data to main ESP8266.

(3) DHT11 was used for monitoring temperature and humidity. The data of the sensors was stored on ThingSpeak cloud and also displayed on the self designed web page hosted on ESP8266 using SPIFFS.

(4) Solenoid valves connected with four channel relay were used to only water the area where needed.

Digital Clock

This was a digital logic design project that was comprised of total of six counters. Two for each hours, minutes and seconds. For each minutes and seconds two counters were used. One counts from 0-9 and other from 0-5 in order to make 59 minutes or 59 seconds. Each counter gets reset after reaching desired number. For hours section one couner counts from 0-1 while other was a 0-9 counter and the combination of two was responsible for 12 hour format. Alarm was also implemented in the clock. Overall the project was very budget freindly and used components like seven segment displays, 555 timer IC, 7493 counter IC, logic gates IC's, bread boards and a 5V power supply.

They have the capability to integrate with microwave circuits and therefore they are very well suited for applications such as cell devices, WLAN applications, navigation systems and many others In this project; a compact rectangular patch antenna was designed using COMSOL software that is used for GPS devices at 1.57542 GHz, and for a satellite TV signal at 11.843 GHz and 11.919 GHz.
